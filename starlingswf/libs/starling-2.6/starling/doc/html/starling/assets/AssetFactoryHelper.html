addPostProcessor()method 
public function addPostProcessor(processor:Function, priority:int = 0):void

Adds a method to be called by the AssetManager when the queue has finished processing. Useful e.g. if assets depend on other assets (like an atlas XML depending on the atlas texture).

Parameters

processor:Function — function(manager:AssetManager):void;
 
priority:int (default = 0) — Processors with a higher priority will be called first. The default processor for texture atlases is called with a priority of '100', others with '0'.
