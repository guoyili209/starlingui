Style Detail
toggleDisplayModestyle
toggleDisplayMode:Boolean

If the displayMode property is set to MediaTimeMode.CURRENT_TIME or MediaTimeMode.REMAINING_TIME, and this property is set to true, the label will switch to displaying the current time and the remaining time, if tapped or clicked. If the displayMode property is not set to one of the specified values, this property is ignored.

The default value is false.

See also

displayMode

displayMode:String

Determines how the time is displayed by the label.

The default value is feathers.media.MediaTimeMode.CURRENT_AND_TOTAL_TIMES.
