fontStylesproperty
fontStyles:FontStylesSet

The internal font styles used to render the text that are passed down from the parent component. Generally, most developers will set font styles on the parent component.

Warning: The fontStyles property may be ignored if more advanced styles defined by the text renderer implementation have been set.


Implementation
    public function get fontStyles():FontStylesSet

stateContextproperty 
stateContext:IStateContext

When the text editor observes a state context, the text editor may change its font styles based on the current state of that context. Typically, a relevant component will automatically assign itself as the state context of a text editor, so this property is typically meant for internal use only.

The default value is null.
