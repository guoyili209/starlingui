Product Version : Feathers 2.1.0

The current version of Feathers. Stable versions will use the format major.minor.patch. Prerelease versions may append additional information after a hyphen. For example, the beta release of Feathers 2.2 would use the following value: 2.2.0-beta. Special builds may append additional information after a plus sign. For example, the stable release of Feathers SDK 2.2 would use the following value: 2.2.0+sdk. The beta version of Feathers SDK 2.2 would use the following value: 2.2.0-beta+sdk.
